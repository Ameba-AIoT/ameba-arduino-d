Follow these steps to apply this patch to run TensorFlow Lite for Microcontrollers on the Realtek RTL8722DM EVBs

1. Copy the contents of the "patch" folder to the Arduino package install location, overwriting any existing files

2. The Arduino package install location can be found at: C:\Users\$USERNAME$\AppData\Local\Arduino15\packages\realtek\hardware\AmebaD\$VERSION$\ where $USERNAME$ is your windows username, and $VERSION$ is the currently installed RTL8722DM Arduino package version.

To revert the patch, in Arduino package manager, uninstall the RTL8722DM package and reinstall.
